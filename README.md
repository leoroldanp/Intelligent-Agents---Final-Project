# Project Martian Pathfinder 
This repository contains the final project developed for the **Design of Intelligent Agents** course (Jan-Jun 2026). The project implements a decision-making agent (Mars Rover) capable of navigating complex, real-world Martian topographical data under strict physical and environmental constraints.

## Features & Implementation
* **Search & Optimization:** Autonomous navigation utilizing the `SimpleAI` framework to implement and compare **A*, BFS, UCS, DFS, Greedy Search, and Simulated Annealing**.
* **Physical Constraints:** Real-time enforcement of maximum climbing slope limits and vertical height difference obstacles.
* **Environment Abstraction:** Full formulation of the agent's **PEAS** (Performance, Environment, Actuators, Sensors) using `mars_map.npy` and `crater_map.npy` datasets.
* **Advanced Visualization:** Interactive 3D route plotting over Martain terrains using `Plotly` and detailed technical analysis.

## Technologies Used
* Python
* SimpleAI
* NumPy & Scikit-Image
* Plotly (Interactive HTML 3D Plots) & Matplotlib
